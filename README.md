# 모두의 학생부종합전형 가이드북 사례분석 몰입 극장

GitHub Pages 업로드용 완성본입니다.

## 파일 구성
- `index.html` : 첫 화면/대학별 메인
- `ewha.html` : 이화여대편
- `kookmin.html` : 국민대편
- `kyunghee.html` : 경희대편
- `uos.html` : 서울시립대편
- `dongguk.html` : 동국대편
- `soongsil.html` : 숭실대편
- `erica.html` : 한양대 ERICA편

## GitHub Pages 업로드
1. 새 저장소를 만듭니다.
2. 이 폴더 안의 파일들을 저장소 루트에 업로드합니다.
3. Settings → Pages → Branch: `main` / folder: `/root` 설정
4. 생성된 주소로 접속합니다.
