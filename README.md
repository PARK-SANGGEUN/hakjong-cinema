# Hakjong Cinema

학생부종합전형 가이드북을 장면처럼 읽는 상담/연수용 몰입형 웹 자료입니다.

## 업로드 방법
1. 이 ZIP 압축을 풉니다.
2. 모든 파일을 GitHub 저장소 루트에 업로드합니다.
4. `index.html`이 첫 화면이며, 각 대학 화면 오른쪽 아래의 **대학별 극장 선택** 버튼으로 첫 화면에 돌아올 수 있습니다.

## 포함 대학
- 이화여대
- 국민대
- 경희대
- 서울시립대
- 동국대
- 숭실대
- 한양대 ERICA


## v4 업데이트
- 각 대학 몰입극장에 `권장과목 보기` 팝업을 추가했습니다.
- 팝업은 이동 가능하고 오른쪽 아래 모서리로 크기 조절이 가능합니다.
- 출처: 2028학년도 권역별 대학별 권장과목(어디가).
